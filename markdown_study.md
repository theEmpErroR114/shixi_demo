# Markdown 文件是什么
Markdown（简称 md）是一种轻量级的标记语言，用简单的符号来表示文本格式（比如标题、加粗、列表、链接等），而不需要像 Word 那样点击按钮排版。文件后缀通常是 .md 或 .markdown，本质上就是一个纯文本文件，可以用任何文本编辑器打开。
